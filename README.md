# Expense Management Project - TypeScript Conversion 

I converted this project from JavaScript to **TypeScript** to improve code safety and structure.

## What I did:
- Moved all logic into `.ts` files in the `src` folder.
- Created an **interface Expense** to manage data objects.
- Defined HTML elements using **Type Assertions** (like `as HTMLInputElement`).
- Configured **tsconfig.json** to output JavaScript files into the `dist` folder.
- Updated HTML files to use `type="module"` for scripts.

## Project Files:
- `src/`: My TypeScript source code.
- `dist/`: The compiled JavaScript files.
- `tsconfig.json`: My TypeScript configuration.
